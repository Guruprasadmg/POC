# Data Set POC

### How to run

Clone the project from
Github -- https://github.com/Guruprasadmg/POC.git
Build and Run:
mvn clean install <br>
mvn spring-boot:run



# Technologies :
Java : 17 <br>
Springboot : 3.3.5 <br>
Database : H2
Build : Maven
